# contextengine-examples

Real `CLAUDE.md`, `.cursorrules`, and `copilot-instructions.md` files generated by [ContextEngine](https://github.com/strifero/ContextEngine) for common project stacks.

Browse these to see what ContextEngine generates before running it on your own project.

## Generate your own

```bash
npx @strifero/contextengine
```

Runs in ~10 seconds. No account. No config. Free.

## How these were generated

Each folder contains the output of running `npx @strifero/contextengine --tool all` against a representative project with that stack's typical config files and dependencies. Files are unedited — raw output. Your project's output will reflect your specific structure and dependencies.
